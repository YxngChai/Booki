# Booki - OpenClassrooms Project

A responsive front-end integration project made as part of the OpenClassrooms Web Developer course.

The goal of this project was to recreate a travel accommodation landing page from a provided Figma design using only HTML and CSS while following modern semantic and accessibility practices.

---

## Preview

Booki is a fictional accommodation booking website inspired by platforms like Airbnb and Booking.com.

The page includes:

- A responsive navigation bar
- Search form
- Filter buttons
- Accommodation cards
- Popular accommodations section
- Activities section
- Responsive footer

---

## Technologies Used

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Font Awesome

---

## Features

- Responsive design for desktop, tablet, and mobile
- Semantic HTML structure
- Accessible forms and navigation
- BEM-inspired class naming
- Hover interactions
- Mobile-first responsive adjustments
- Custom favicon support

---

## Accessibility Improvements

This project includes several accessibility practices:

- Semantic landmarks (header, nav, main, section, footer)
- Accessible form labels
- Decorative icons hidden from screen readers using aria-hidden
- Meaningful image alt text
- Keyboard-focusable links
- Internal page navigation with anchor links

Safari accessibility audit passes successfully.
